Pagina Web en Wordpress

Los desarrollos basados en Wordpress son Páginas Web Administrables con un panel administrativo con las opciones que el mismo dispone. Este tipo de proyecto es realizado por nosotros según las especificaciones del cliente y suponiendo que necesite que el desarrollo sea en esta plataforma.

Con este tipo de proyecto usted tendrá su página web altamente profesional y dinámica: Usted ó la persona que manejará su sitio web, podrá editar sus contenidos desde un panel administrativo de manera rápida, sencilla y sin depender de un experto ni dinero.

Detalles del Sitio Web

  • Diseño de Sitio Web usando Colores y Gustos dados por el cliente.
  • Panel de acceso para Administradores del Sitio Web
  • Link principal o Portada del Sitio Web
  • Links en Menu organizado según el Panel Administrativo
  • Contenido en la pagina basado en texto, imágenes o videos (Incrustados desde Youtube) .
  • Categorizacion de su contenido en la pagina web
  • Link de Contacto con su Formulario para envio de Mensajes
  • Banners Publicitarios.
  • Link a redes sociales como twitter, Facebook, Instagram.
  • Galería fotográfica con imágenes relacionadas con su sitio web
  • Configuración de Palabras Clave en su Web para Google y otros Buscadores
  • Agregar archivos PDF, DOCX entre otros
  • Visible desde Telefonos Celulares a través de un Diseño Responsitivo

Panel Administrativo

El panel de acceso es el área desde la cual se podrá Administrar las áreas de su sitio web.
  • Acceso a Panel de control de la Pagina Web
  • Opción de Agregar y Editar contenido en la página web.
  • Agregar al Contenido archivos PDFs, DOCs, entre otros.
  • Opciones de crear categorías en la pagina para organizar contenidos
  • Crear y cambiar imágenes de banners publicitarios dentro de la pagina web
  • Creación de galerías de fotos .
  • Área de Contacto con su Formulario para envió de Mensajes
  • Gestión de Mensajes recibidos desde su Página Web.
  • Configuración de Redes Sociales.
  • Crear su propio Menu principal con sus correspondientes links
  • Agregar Palabras Claves de su sitio web para Google.
  • Gestion de Administradores de la Pagina Web.
